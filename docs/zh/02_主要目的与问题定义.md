# 主要目的与问题定义

## 主要目的（Objectives）

- **提供可信信息**：为国际学生提供 **准确、可追溯、可更新** 的生活信息与攻略。
- **降低适应成本**：减少初来乍到的 **困惑、压力、试错成本与孤立感**。
- **构建互助社区**：鼓励学生贡献真实经验（住房、交通、购物、避坑），形成“可复用的经验库”。
- **AI + 人类协作**：用 AI 解决“快速问答与信息检索”，用人类解决“校验、语境与责任”。

## 现状问题（Problem）

很多平台的内容混杂：

- **有价值信息与广告混在一起**：需要用户自行过滤软广、带货、商业推广。
- **建议可能被营销动机扭曲**：推荐与真实体验不一致，存在偏见或利益冲突。
- **新生信息不对称更严重**：对住房、签约、交通、诈骗识别等缺乏经验，容易踩坑。

对初到美国的国际学生，这会造成：

- **误导性住房信息**：虚假房源、诱导转账、合同陷阱、治安风险。
- **不可靠攻略**：过期流程、误解政策、与地区/学校不匹配。
- **心理与社交压力**：没有可信的人可以问，孤立感与焦虑上升。

## 解决方案概述（Solution）

搭建一个专为国际学生设计的网站，具备：

- 交通指南、城市生活攻略、传统食材/超市信息
- 学生真实住房体验与避坑分享
- 诈骗与安全意识教育（案例库 + 清单）
- 安全的线上志愿者连接（受控流程、隐私保护）
- AI 快速问答（基于已审核内容检索，并标注来源与可信状态）

## 成功的定义（Success）

- **信息可信**：重要内容可追溯来源，并有清晰的“验证状态”与“更新时间”。
- **用户省时省心**：用户能更快完成关键决策（找房、通勤、购物、办事）。
- **社区可持续**：真实贡献者与志愿者能持续参与，内容持续更新与纠错。
- **安全可控**：诈骗/骚扰/垃圾内容被有效控制，用户隐私默认受保护。
